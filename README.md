# Installation

Clone project

```shell
git clone https://github.com/lexxyar/beck-depression-inventory.git
```

Go to project folder

```shell
cd beck-depression-chart
```

Install dependencies

```shell
npm install
```

Run development server

```shell
npm run dev
```

Enjoy it!
